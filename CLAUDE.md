## Game design

The central game design documentation lives in `game-doc/`, a Docusaurus site — everything in this engine is built to run this game. Read it whenever a task is about the game itself (mechanics, content, balance, features driven by game design) rather than pure engine architecture, and keep it updated as the design evolves.
